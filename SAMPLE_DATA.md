## How to Use This Data

1. **Open Unity** and wait for scripts to compile

2. **Create the Traits first** (you'll need them to reference in profiles and posts):
   - Navigate to `Assets/Data/ScriptableObjects/Traits/Interests/`
   - Right-click â†’ Create â†’ Maskhot â†’ Traits â†’ Interest (create all 5 interests)
   - Repeat for Personality Traits and Lifestyle Traits in their respective folders
   - Fill in the data from the sections above

3. **Create Candidate Profiles**:
   - Navigate to `Assets/Data/ScriptableObjects/Profiles/`
   - Right-click â†’ Create â†’ Maskhot â†’ Candidate Profile
   - Fill in basic info (name, gender, age, bio, archetype)
   - **Drag trait assets** into the personality traits, interests, and lifestyle traits arrays

4. **Add Guaranteed Posts** (in the same profile asset):
   - Scroll to "Social Media Posts" section
   - Click the **+** button under "Guaranteed Posts" to add a post
   - Fill in post data (type, content, timestamp, likes, comments)
   - **Drag trait assets** into the "Related Interests", "Related Personality Traits", and "Related Lifestyle Traits" arrays
   - Note: You can leave these arrays empty if the post doesn't relate to specific traits (like the red flag posts)
   - Check the appropriate flags (Is Red Flag / Is Green Flag)

5. **Test in Play Mode** to verify everything works!
